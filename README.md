# http://microcovid.org

## Local Development

Requirements:

- Node.JS 12
- Yarn

(See `.tool-versions` for specific and up-to-date versions).

1. Install local depenendences
    ```sh
    $ yarn install
    ```
1. Start local server
    ```sh
    $ yarn start
    ```
1. Open http://localhost:3000
